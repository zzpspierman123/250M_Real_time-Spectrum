## ⭐️About Project

**该项目制作了一个250M采样率的频谱仪，能够对125M以内的数据进行傅里叶分析，其精度达到100k。** 
<div>
 
***250M采样采样率*** 通过FPGA的AD模块，以250M采样率采集信号并通过FIFO传给UART。
<div>

***数据分析：*** STM32H750通过UART接收数据，并进行基4的傅里叶分析并打出频谱。
<div>
